数据队列初始化

调用该函数将初始化数据队列控制块，并设定通知回调函数。

参数
queue数据队列控制块
size数据队列块大小
lwm阈值设定
evt_notify通知事件回调函数
返回
-RT_ENOMEM 内存不足申请失败, RT_EOK 初始化成功

向数据队列中压入数据

调用该函数可以向指定的数据队列中压入数据，当数据队列已满，则在指定的超时时间内挂起等待。

参数
queue数据队列控制块
data_ptr待压入的数据指针
data_size待压入数据的字节数
timeout指定超时时间
返回
-RT_ETIMEOUT 队列已满，等待超时；RT_EOK 压入成功
