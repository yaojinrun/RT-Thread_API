管脚工作模式设置

此函数可以设定指定管脚的工作模式。

参数
pin管脚编号
mode管脚工作模式，可以去以下 5 种模式之一
参数 说明
PIN_MODE_OUTPUT 输出，具体模式看 drv_gpio.c 源码实现
PIN_MODE_INPUT 输入
PIN_MODE_INPUT_PULLUP 上拉输入
PIN_MODE_INPUT_PULLDOWN 下拉输入
PIN_MODE_OUTPUT_OD 开漏输出

rt_err_t rt_pin_attach_irq ( rt_int32_t  pin,
rt_uint32_t  mode,
void(*)(void *args)  hdr,
void *  args 
)

绑定管脚中断

此函数可以设定指定管脚的中断出发模式，并可指定中断回调函数。

参数
pin管脚编号
mode中断触发模式，可取以下 5 种值之一
参数 描述
PIN_IRQ_MODE_RISING 上升沿触发
PIN_IRQ_MODE_FALLING 下降沿触发
PIN_IRQ_MODE_RISING_FALLING 边沿触发（上升沿和下降沿都触发）
PIN_IRQ_MODE_HIGH_LEVEL 高电平触发
PIN_IRQ_MODE_LOW_LEVEL 低电平触发
hdr中断回调函数，用户需要自行定义这个函数，其返回值为 void
args中断回调函数的参数，不需要时设置为 RT_NULL
返回
RT_EOK 成功；RT_ENOSYS 无系统；RT_EBUSY 忙
