详细描述

内核对象管理接口

内核对象系统可以访问和管理所有内核对象。

内核对象包括内核中的大多数设备：

  • 线程
  • 信号量和互斥量
  • 事件/快速事件，邮箱，消息队列
  • 内存池
  • 定时器
    Figure 2: Kernel Object
    内核对象可以是静态对象，其内存在编译时就已分配。 也可以是动态对象，其内存是在运行时从系统堆中分配。
